
**The use of EukPhylo version 1.0 in metatranscriptomics.** 
EukPhylo is a phylogenomic toolkit for processing transcriptomic and genomic data that includes novel phylogeny-informed contamination removal techniques. See our Wiki (https://github.com/Katzlab/EukPhylo/wiki/metatranscriptomics) for more information on installation and usage!
